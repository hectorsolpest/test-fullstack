## Ejemplo de uso de ramas en git

- Crear la rama 
```console
git branch feature-xyz
```

- Cambiar de rama
```console
git checkout feature-xyz
```

- Añadir cambios
```console
git add .
git commit -m "Añadir cambios en feature-xyz"
```

- Cambiar a la rama principal
```console
git checkout main
```

- Fusionar la rama 
```console
git merge feature-xyz
```
